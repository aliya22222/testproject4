## Описание программы
Данное приложение — это графический генератор случайных паролей, разработанный с помощью языка Python и библиотеки Tkinter. Программа позволяет пользователю задавать параметры генерации пароля, такие как длина и типы символов (цифры, буквы, спецсимволы). Все сгенерированные пароли автоматически сохраняются в историю, которая загружается при запуске программы и хранится в файле JSON. Это удобно для хранения и последующего использования безопасных паролей.

## Возможности
- Выбор длины пароля с помощью ползунка (от 4 до 32 символов).
- Включение/выключение использования цифр, букв и специальных символов через чекбоксы.
- Генерация случайного пароля по выбранным параметрам нажатием кнопки.
- Автоматическое сохранение сгенерированных паролей в историю.
- Просмотр истории ранее сгенерированных паролей в таблице.
- Загрузка истории при запуске программы.
- Простая и интуитивно понятная графическая оболочка.

## Примеры использования

### Генерация пароля длиной 12 символов с использованием цифр и букв

Установите ползунок длины на 12.
Отметьте чекбоксы «Цифры» и «Буквы».
Нажмите кнопку «Генерировать».
В поле появится сгенерированный пароль, например:

          aB3dE5fGh6I2
      
Этот пароль автоматически сохранится в историю и появится в таблице ниже.

### Генерация пароля длиной 8 с использованием только спецсимволов

Установите длину 8.
Отметьте только «Спецсимволы».
Нажмите «Генерировать».
В поле появится пароль, например:

          !@#$%^&*
      
Пароль сохранится в историю.
